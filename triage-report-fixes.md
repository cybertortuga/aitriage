# triage-report-fixes

## Обзор
We need to fix rendering, formatting, and date bugs in the AI triage agent reports:
1. The Markdown tables generated by the LLM in sections 3, 4, 5, and 6 are broken in the GitHub Action summary due to a missing header separator line (`| --- | --- |`).
2. The report date generated by the LLM defaults to "October 26, 2023", which is outdated. The actual date should dynamically reflect the current date (June 16, 2026).
3. Emojis are forbidden in all parts of the workflow names, reports, and communication.
4. Truncations (such as incomplete sentences ending in `...` or `e. |`) must be prevented.

## Затронутые файлы и области
- `internal/agent/graph/orchestrator.go` (Go source code for prompt formatting and report generation)
- `internal/agent/prompts/templates.go` (Go source code containing LLM system/user prompts)
- `cybertortuga/aitriage-test-vulnerabilities` workflow runs (Verification target)
